
#  Customer Behavior Analysis
## Project Overview
This project focuses on analysing  Customer Behavior to uncover insights, trends, and patterns that can support better decision making.  
Using tools such as Python, SQL and visualization libraries, this analysis explores key metrics, performs data cleaning, and builds dashboards where required.
##  Objectives
- Understand the structure and quality of the data  
- Clean and preprocess the dataset  
- Perform exploratory data analysis (EDA)  
- Identify trends, correlations, and outliers  
- Build visualizations to represent insights clearly  
- Provide actionable recommendations  
---
##  Dataset Description
Dataset contain 3900 rows with customer details,
Product details and the purchase details

##  Conclusion
-	Boost Subscriptions – Promote exclusive benefits for subscribers. 
-	Customer Loyalty Programs – Reward repeat buyers to move them into the “Loyal” segment. 
-	Review Discount Policy – Balance sales boosts with margin control. 
-	Product Positioning – Highlight top-rated and best-selling products in campaigns. 
-	Targeted Marketing – Focus efforts on high-revenue age groups and express-shipping users.
